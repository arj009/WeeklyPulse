# Raw exports (`data/raw/`)

Place **public** App Store and Play Store review export files here before each weekly run.

- This folder is **gitignored** — exports must not be committed.
- Download steps: see Phase 2 operator guide (to be added in `docs/operator-export-guide.md`).
- Supported formats documented in Phase 2.

Example filenames (your exports may differ):

- `app_store_reviews.csv`
- `play_store_reviews.csv`
